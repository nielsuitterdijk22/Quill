# Sign up & your first repo

You don't need an organization to get started. Sign up, and you immediately have a personal namespace at `quill.so/your-username` where you can create repos.

## Sign up

Go to [quill.so](https://quill.so) and create an account. You can sign up with GitHub or with an email and password.

Once you're in, you land on your personal dashboard.

## Create your first repo

Click **New repository** from the dashboard or from the **+** menu in the top nav.

Fill in:
- **Name** — becomes the URL slug. Use lowercase letters, numbers, hyphens, and underscores. Keep it short.
- **Description** — optional, but useful.
- **Visibility** — Private (just you), Internal (all org members, once you're in an org), or Public (anyone).

Click **Create repository**.

Your repo is live at `quill.so/your-username/repo-name`.

## Initialize with a README

If you check **Initialize this repository**, Quill commits a `README.md` to the default branch for you. Good if you're starting fresh.

If you're pushing an existing local repo, leave it unchecked — you'll push your own history.
